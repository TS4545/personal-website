Railway track Visualizer

website that uses an interactive, custom-styled Google Map to highlight major North American freight train routes and allows users to search for cities to instantly zoom to rail hubs. It's built using HTML, JavaScript, and Tailwind CSS for styling, and uses GitHub Actions for secure deployment to hide the necessary mapping key.

Civly Social Media

Civic engagement based social media platform that allows community members to submit local issues, receive automated AI-generated summaries, and access relevant legislative context in an intuitive interface. The system is built using a React and Tailwind frontend, an Express backend with SQLite, and integrated OpenAI API for analysis and information delivery.

View Live Site | Github Repo

RC Aviation rocket gyroscope and website demo

Embedded flight-control system that uses an onboard gyroscope module to capture real-time orientation and angular velocity data, enabling precise tracking of a rocket’s attitude throughout flight. Developed in Arduino C++, the code processes gyroscopic telemetry, and triggers parachute deployment at the correct stage to ensure a safe and reliable recovery sequence.
